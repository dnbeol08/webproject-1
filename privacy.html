개인정보처리방침

최종 수정일: 2026년 2월 27일

Pracora는 가능한 한 많은 기능을 브라우저 안에서 처리하고, 사용자가 입력한 데이터를 운영자가 별도 데이터베이스에 장기 저장하지 않는 방향을 기본 원칙으로 삼습니다. 다만 광고, 방문 통계, 문의 응대처럼 서비스 운영에 필요한 범위에서는 외부 서비스가 제한적으로 데이터를 처리할 수 있습니다.

1. 브라우저 내 입력 데이터

메인 페이지의 요약기, 단위 변환기, 비밀번호 생성기, 키워드 추출기, 답장 초안 기능은 기본적으로 사용자의 브라우저에서 실행됩니다. 입력한 텍스트와 숫자는 운영자가 별도로 수집하거나 회원 DB에 저장하지 않습니다.

2. 방문 통계 및 행동 분석

본 사이트는 Google Analytics와 Microsoft Clarity를 사용해 방문 수, 페이지 흐름, 클릭 위치, 이탈 경향 등 익명 또는 가명 수준의 사용 행태를 분석할 수 있습니다. 이 데이터는 UI 개선과 콘텐츠 구조 개선에 사용합니다.

3. 광고 서비스

Google AdSense 등 제3자 광고 서비스는 맞춤형 광고, 빈도 제한, 사기 방지, 측정 목적 등을 위해 쿠키 또는 유사 기술을 사용할 수 있습니다. 광고 관련 데이터 처리는 각 사업자의 정책을 따릅니다.

4. 문의 시 제공 정보

사용자가 이메일로 문의할 경우, 회신을 위해 발신 주소와 문의 내용이 이메일 서비스 제공자에 저장될 수 있습니다. 해당 정보는 문의 답변과 기록 관리 목적 외에는 사용하지 않습니다.

5. 아동 개인정보

본 사이트는 아동을 주요 대상으로 설계된 서비스가 아닙니다. 보호자가 아동의 개인정보 제공이 부적절하다고 판단하는 경우 문의 페이지를 통해 삭제 요청을 보낼 수 있습니다.

6. 외부 링크

사이트 안의 외부 링크를 클릭하면 해당 외부 사이트의 정책이 적용됩니다. 외부 사이트의 데이터 처리 방식은 Pracora가 통제하지 않습니다.

7. 정책 변경

운영 도구, 광고 방식, 문의 수단이 변경될 경우 이 문서를 함께 갱신합니다. 중요한 변경은 페이지 상단의 최종 수정일에 반영합니다.

문의

개인정보 관련 문의는 문의 페이지를 통해 보내실 수 있습니다.
