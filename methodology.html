도구 설명 원칙

최종 수정일: 2026년 2월 27일

Pracora의 각 도구는 가능한 한 단순하고 설명 가능한 방식으로 동작합니다. 사용자는 도구가 무엇을 잘하고 무엇을 보장하지 못하는지 이해한 상태에서 활용할 수 있어야 합니다.

1. 요약기

입력 문장을 분리한 뒤, 길이와 핵심 키워드 포함 여부를 기준으로 우선순위를 계산합니다. 이는 의미 이해형 대형 모델이 아니라 규칙 기반 압축 보조에 가깝기 때문에, 세밀한 뉘앙스나 맥락은 직접 검토해야 합니다.

2. 단위 변환기

표준 환산식을 사용합니다. 반올림 자릿수는 가독성을 위한 설정이며, 고정밀 계산이 필요한 경우 원시 수식을 다시 검산하는 것이 좋습니다.

3. 비밀번호 생성기

브라우저의 crypto.getRandomValues를 사용해 예측 가능성을 낮춥니다. 생성된 비밀번호의 보안 수준은 저장 방식, 재사용 여부, 2단계 인증 사용 여부에도 크게 좌우됩니다.

4. 키워드 추출기와 답장 초안

둘 다 초안 작업을 빠르게 만드는 용도입니다. 키워드 추출은 빈도 기반으로 중요한 표현을 가려내고, 답장 초안은 톤 템플릿을 적용해 응답 문장을 제안합니다. 민감한 문맥에서는 최종 수정을 거쳐야 합니다.

핵심 원칙

Pracora는 결과의 신비화보다 설명 가능한 동작과 검토 가능성을 더 중요하게 봅니다.
